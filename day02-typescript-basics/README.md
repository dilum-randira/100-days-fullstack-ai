Today I practiced core TypeScript fundamentals by setting up a small Node + TypeScript project and implementing a few simple, strongly-typed utility functions.

## What I Practiced

- Basic TypeScript types (`number`, `string`, `boolean`)
- Object types with optional properties
- Generic functions with type parameters (`<T>`)
- Union types (`number | string`)
- A simple project structure using `tsconfig.json`, `src`, and `dist`

## How to Run This Project

1. Install dependencies:
   ```bash
   npm install
   ```

2. Build the TypeScript code:
   ```bash
   npm run build
   ```

3. Run the compiled JavaScript:
   ```bash
   npm start
   ```

You should see console output for each of the example functions from `src/utils.ts`.
