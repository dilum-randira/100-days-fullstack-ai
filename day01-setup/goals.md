# Challenge Goals

- Build full-stack projects using the **MERN** stack.
- Strengthen **TypeScript** skills across frontend and backend.
- Practice building **React** components and small applications.
- Learn and apply **Node.js** and **MongoDB** for real-world APIs.
- Explore **Python AI/ML** fundamentals with practical mini-projects.
- Experiment with **OpenCV** for basic computer vision tasks.
- Try **TensorFlow.js** for in-browser machine learning demos.
- Maintain daily progress and notes for all 100 days.
